# cinatra-ai/dev

Claude Code skills that help a single developer set up and build **with**
Cinatra. Install once; the skills activate on natural-language triggers.

> Licensed under the Apache License 2.0 — see [LICENSE](./LICENSE).

## Install

```sh
npx @cinatra-ai/dev --claude --global
```

## Skills

| skill | what it does |
|---|---|
| `setup` | bootstrap a fresh contributor machine (toolchain + Claude baseline) |
| `onboarding` | first-run orientation for a new Cinatra contributor |
| `cinatra-dev-env` | run the local dev + verify stack |
| `extension-conventions` | conventions for building a Cinatra extension |
| `domain-gotchas` | domain pitfalls to avoid while building |
